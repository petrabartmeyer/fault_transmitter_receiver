This repository contains the results obtained in the case study of the article "Optimized Allocation of Fault Transmitters for Improving the Recloser-Fuse Coordination".

There is a specific folder for each test system with the results obtained in Sec. V.A (without DER) and Sec. V.B (with DER).

Legend for File Name:
	
	test_systemA-BFT_woDER.pdf - results without DER;
	test_systemA-BFT_wDER.pdf  - results with DER;

	A - Number of buses
	B - Number of fault transmistters
